浏览器缓存

发表于 2019-09-21 更新于 2021-03-21 分类于 前端

什么是浏览器缓存？

为了节约网络的资源，浏览器在用户的磁盘上对最近请求过的文档进行存储。当再次访问时，就可以直接从本地磁盘中获取，这样就可以加速页面的阅览，并且缓解一部分服务器的压力。
浏览器缓存的缓存机制是通过 Http 协议来实现的。

浏览器缓存存在哪？

浏览器缓存的位置可以分为四种：
1.Service Worker
2.Memory Cache
3.Disk Cache
4.Push Cache
